## 操作方法


- 左矢印キーまたはA：左に移動
- 右矢印キーまたはD：右に移動
- スペースキー：ジャンプ

### Assets/Scripts/StateMachineExample.cs

- StateMachineBehaviour を継承したサンプルコード
  - 各Callback関数が呼ばれるとデバッグログ出力する
  - Jump にアタッチしている
